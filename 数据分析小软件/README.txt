【使用步骤】

1. 点击"导入数据文件"按钮
   - 选择 test_data.csv（或其他 CSV/Excel 文件）
   - 程序会自动读取数据信息

2. 选择因变量(Y)
   - 从下拉框中选择一个因变量（如"销售额"）

3. 选择自变量(X)
   - 在列表中可多选（Ctrl+点击）多个自变量
   - 如：广告投入、促销次数、时间成本

4. 点击"执行线性回归分析"按钮
   - 程序自动计算回归模型

5. 查看结果
   - 分析结果：详细的统计输出
   - 模型汇总表：R²、F值等核心指标
   - 系数表：每个变量的系数和P值
   - 方差分析表：ANOVA结果

Q: 为什么程序启动很慢？
A: PyQt6首次启动会初始化GUI环境，属于正常现象，耐心等待即可。

Q: 能否导入我自己的数据？
